# Frontend Overview

This section groups the documents that explain how the React frontend is used
and how it is structured.

## Documents

- [Usage Guide](usage-guide.md): manual validation flow and explanation of the
  visible operations.
- [Technical Notes](technical-notes.md): build, runtime, and source layout
  details for the frontend layer.

## Scope

The frontend exists as a compact operator workbench. It is intentionally small
and keeps the focus on validating the API integration with the Rust chain
service rather than on building a large client application.
